This is a Discord Bot that is programmed using Node.js using JavaScript. This bot has the capability to access Discord and OpenAI's APIs in order for users to interact with ChatGPT from Discord.
The bot utilizes the GPT-4 model all users need to do is @mention the bot and they can chat and ask questions. The bot also has the functionality of interacting with other GPT bots if the same code is duplicated
which makes for an interesting conversation between AI.
